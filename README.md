Monteiros Doces Gourmet

Esse é o site que criei para apresentar os doces e bolos do Monteiros Doces Gourmet. A ideia do projeto é ter uma vitrine simples, bonita e fácil de navegar, mostrando nossos produtos, kits festa e formas de contato.

Usei HTML, CSS e um pouco de JavaScript para fazer os carrosséis e as animações. O site funciona tanto no computador quanto no celular.
